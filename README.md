DNA-MUT
============
This simple C# consol application that can be used to do a simple task of taking two random SEQ of Nucleic Acid "equal in length" and Make THIS magic (:D) :-

1- Count number of Nucleotides.

2- Check similarity or difference and highlighting it.

3- calcualted the percent of similarity and difference.

4- Indecate the index of each Nucleotides and compare between the orginal and mutated.

How To Use:-
=================
(1)----> Enter your "Unknown/extracted SEQ".

(2)----> Enter your "Normal/Control SEQ".

(3)----> Enter "1" for similarity and "2" for difference.
