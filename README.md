DNA-MUT
============
This simple C# program that can be used to do a simple task of taking two random SEQ of Nucleic Acid "equal in length" and Make THIS magic (:D) :-
\n1- Count number of Nucleotides.
\n2- Check similarity or difference and highlighting it.
\n3- calcualted the percent of similarity and difference.
\n4- Indecate the index of each Nucleotides and compare between the orginal and mutated.
